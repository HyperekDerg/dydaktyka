# Zbiór zadań

## Referencje

**Zadanie 1.** Uzupełnij poniższy kod źródłowy:
```cpp
#include <iostream>

// Tutaj powinna znaleźć się definicja funkcji f.

int
main()
{
  int n = 0;
  f(n) = 42;
  std::cout << n << '\n'; // Wypisze wartość: 42.
}
```
Uzupełniony program powinien wypisywać następujący komunikat:
> 42
